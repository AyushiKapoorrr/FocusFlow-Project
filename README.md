# FocusFlow-Project

FocusFlow is a simple Python-based productivity tracking app that lets users add tasks, monitor their progress, visualize their daily routines, and understand how efficiently they spend their time. Built using **CustomTkinter**, **Matplotlib**, and **CSV storage**, it provides a clean interface and helpful analytics for improving focus and time management.

## **Features**

* Add, view, update, and delete tasks
* Track hours, categories, and completion status
* View productivity statistics
* Visual charts (bar, pie, line)
* Automatic data storage using CSV
* Simple GUI dashboard using CustomTkinter

## **Run the App**

```bash
python main.py
```
